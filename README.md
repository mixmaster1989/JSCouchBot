# JSCouchBot (RU/EN)

RU
---
Телеграм‑бот для обучения JavaScript: карточки/вопросы, мини‑упражнения, сохранение прогресса. Подходит для EdTech‑сценариев (самообучение, поддержка учеников).

Быстрый старт:
```bash
cp .env.example .env
npm i && npm run start
```
